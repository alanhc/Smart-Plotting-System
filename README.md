# Smart-Plotting-System
本專案為全國微電腦創作競賽 2018的比賽專案，構想來自年長者喜歡種植植物，因此使用Linkit7697為開發版及多個感測器模組得知天氣資訊，並有中斷處理的傾倒偵測、藍芽配對及結合自然語言及盆栽擬人化的手機APP介面可以與之對話，使年長者與盆栽互動增加趣味性。
